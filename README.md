# Student-Placement-Prediction-ML
Machine learning project for predicting student placement using classification algorithms and performance evaluation.
## 🚀 Project Overview
This project focuses on analyzing factors influencing student placement. We addressed imbalanced data using **SMOTE** and applied **Scaling** techniques to ensure high accuracy for our classification models.

## 🛠 Tech Stack
- **Python**: Core programming language.
- **Scikit-learn**: For building and training machine learning models.
- **Pandas & NumPy**: For data manipulation and analysis.
- **Matplotlib & Seaborn**: For data visualization.

## 📊 Key Results
After evaluating multiple models (SVM, Random Forest, Logistic Regression, etc.), we concluded that:
* **Logistic Regression** provided the best balance and performance for this dataset.
* Emphasis was placed on improving the **Recall** metric to ensure robust prediction for the minority class.

## 📁 Project Structure
- `dataset/`: Contains the `Internship_Selection_Dataset.csv` file.
- `Student_Placement_Prediction.ipynb`: The complete notebook for analysis and modeling.
